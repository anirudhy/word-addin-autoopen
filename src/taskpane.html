Approach #1: User-Controlled Auto-Open

This approach allows users to enable/disable auto-open behavior using Office.js settings API. The setting is stored per-document and persists across sessions.

How It Works

  1. User clicks "Enable Auto-Open" button
  2. Code sets Office.AutoShowTaskpaneWithDocument = true
  3. Setting is saved to the document
  4. Next time document opens (and add-in is installed), task pane opens automatically
  5. WebSocket connection establishes when pane opens
